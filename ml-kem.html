Product Milestone

ML-KEM Complete Subsystem RTL
Full FIPS 203 Compliance Achieved

May 2026  ·  Xilinx 7-Series Target

We are proud to announce the complete RTL readiness of our ML-KEM-512 hardware core, delivering full cryptographic loop compliance with the final FIPS 203 standard (Module-Lattice-Based Key-Encapsulation Mechanism). All processing stages—Key Generation, Encapsulation, and Decapsulation—are completed, structurally optimized, and verified via unified end-to-end simulation pipelines.

Production Milestone: The complete cryptographic loop is operational. The hardware subsystem seamlessly ingests initial seeds, distributes public/private keys, handles ciphertext evaluation, and enforces definitive shared-secret verification.

Design Philosophy: Minimum Footprint

Vicip IP cores are designed specifically for low-resource, low-power targets — embedded systems, IoT devices, and edge hardware where silicon area and power consumption constraints are strict.

Key architectural constraints

A single, unified DSP/multiplier structure handles all polynomial modular reductions across the entire design using an optimized Montgomery math engine. Internal BRAM memory mapping is tightly constrained. Footprint minimization is solved directly at the microarchitectural pipeline level rather than relying blindly on aggressive synthesis flags.

SoC-Ready Integration: APB & AXI4-Stream

To guarantee effortless dropping into standard SoC processing nodes, the accelerator features an isolated dual-bus control plane and data plane topology. System commands, configuration, and status monitoring are mapped via an ultra-low-gate APB slave register interface. High-bandwidth cryptographic payloads—seeds, public keys, ciphertexts, and verified shared secrets—are continuously piped via dedicated AXI4-Stream Master and Slave interfaces backed by integrated asynchronous FIFO buffering.

Implementation Results

Following the deployment of targeted point-to-point asynchronous path constraints to manage internal clock domain boundaries, the subsystem achieves timing closure flawlessly. The design locks in at a 83.33 MHz (12ns) clock rate on standard Xilinx 7-Series silicon, achieving optimal throughput-to-area performance.
